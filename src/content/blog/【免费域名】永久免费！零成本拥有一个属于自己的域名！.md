## 免费域名注册与订阅提醒系统搭建指南
:::note{type="success"}
 本服务永久免费，但需每180天续订一次（续订免费），续订后可再用365天。当剩余天数不足180天时可再次续费。
 
 **总花费：0元人民币**
:::
---

### 第一步：注册免费域名

#### 1. 查询域名可用性
:::note
访问 **[DPDNS 官网](https://dpdns.org)**，在搜索框中输入想要的域名。

**注意**：域名后缀必须为 `.dpdns.org` 或 `.qzz.io`（目前仅这两种免费）。
:::
若域名可用，会显示如下界面：
![域名可用提示](https://oss3-bbs.mt2.cn/forum/202510/04/223629c0vfkcqa0dkuwfji.jpg)

#### 2. 注册账号
前往 **[注册页面](https://dash.domain.digitalplat.org/auth/register)** 填写信息：
:::note{type="info"}
- **用户名**、**邮箱（需常用）**、**密码** 必填
- 其他信息可随意填写（如手机号位数正确即可，地址可随机生成）
:::
![注册页面](https://oss3-bbs.mt2.cn/forum/202510/04/224050ufabni7z9rjkpy7p.jpg)
:::note
注册后需**验证邮箱**，验证成功后即可登录。

**福利**：授权 GitHub 并为项目点 Star，可额外获得一个免费域名（也就是说每个账号最多两个）。
:::
#### 3. 登录并注册域名
登录后进入 **[仪表盘](https://dash.domain.digitalplat.org/)**，点击侧边栏菜单中的 **Domain Registration**。
:::note
输入域名前缀（后缀选择 `.dpdns.org`或`.qzz.io`），点击创建即可。**保留此页面备用**。
:::
---

### 第二步：托管域名到 Cloudflare

#### 1. 添加域名到 Cloudflare
登录 **[Cloudflare 控制台](https://dash.cloudflare.com/login/?lang=zh-hans-cn)**，点击「添加站点」，输入你的完整域名（如 `xiao313.dpdns.org`），选择免费套餐。

#### 2. 获取 NS 记录
:::note
在 Cloudflare 概述页面下滑，找到两个 NS 记录地址，复制并填入第一步中保留的域名注册商页面（Name Servers 1 和 2），保存设置。
:::
![NS 记录示例](https://oss3-bbs.mt2.cn/forum/202510/04/230018sdfy7uy7f7fknarf.jpg)

#### 3. 等待生效
:::note
返回 Cloudflare，稍等几分钟直至状态变绿，即表示托管成功。后续可通过 Cloudflare 添加 DNS 记录解析项目。
:::

### 第三步（可选）：部署订阅提醒系统（防止忘记续费）

### 1. 下载并部署 Worker
下载订阅提醒脚本：
[下载方式1（优先选）](https://wwwj.qyshare.com:2999/s/iip57i)

[下载方式2](https://intl.ozabc.com/page/12.html)
:::note
在 Cloudflare 控制台进入 **Workers & Pages**，创建新 Worker，选择「上传文件」并上传刚下载的 `index.js`。
:::
#### 2. 配置 KV 数据库
:::note
- 进入 **存储和数据库 → KV**，创建命名空间，**名称必须为 `SUBSCRIPTIONS_KV`**。
- 回到 Worker 的「设置」→「绑定」，添加 KV 命名空间绑定：
  - **变量名称**：`SUBSCRIPTIONS_KV`
  - **KV 命名空间**：选择刚创建的 `SUBSCRIPTIONS_KV`

#### 3. 绑定域名并设置路由
在 Worker 的「触发器」中绑定你的域名（如 `sub.xiao313.dpdns.org`），系统将自动创建子域。

访问该地址，使用默认账号登录：
- 用户名：`admin`
- 密码：`password`
:::
:::note{type="warning"}
**登录后请立即修改密码！**
:::
#### 4. 设置定时执行
:::note
在 Worker 的「设置」→「触发器」中添加 Cron 触发器，设置为每天中午12点执行（如 `0 12 * * *`）。
:::

### 第四步：配置通知渠道（多端提醒）

#### 1. Telegram 通知
:::note
- 通过 [@BotFather](https://t.me/BotFather) 创建机器人，获取 API Key。
- 使用 [@VersaToolsBot](https://t.me/VersaToolsBot) 获取你的 Chat ID（转发一条消息即可）。
- 将 API Key 和 Chat ID 填入订阅管理系统。
- **记得先向机器人发送 `/start`**。

![Telegram Chat ID 示例](https://oss3-bbs.mt2.cn/forum/202510/04/232531wcssx888koxfo8bx.jpg)

#### 2. NotifyX 通知（微信提醒）
- 访问 [NotifyX](https://notifyx.js.org/)，微信扫码关注服务号，自动获取 API Key。
- 将 API Key 填入系统即可。

#### 3. 邮箱通知（可选自定义域名）
- 推荐使用 **Resend** 服务，注册后获取 API Key。
- 发件人邮箱可自定义（自己设置，要添加对应的DNS记录），或使用默认（见下方）
- 将 API Key 和邮箱地址填入系统，建议接收邮箱使用 QQ 邮箱。

发件邮箱地址示例（默认）：
```text
xxxx@resend.dev
```
:::
