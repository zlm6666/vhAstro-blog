## 部署教程
:::note
1. 安装[GitHub King](https://bbs.binmt.cc/thread-159973-1-1.html) 这个主要是用来更方便的管理文章、文件的（需要GitHub token，软件里有教程）
2. 在GitHub King中搜索`vhAstro-Theme`，长按克隆到自己仓库
3. 回到自己仓库，编辑一些代码（不然的话部署好的就是别人的博客界面）
打开配置文件 `src/config.ts` ，按照以下提示编辑里面的内容（直接编辑就可以，不要删）

│   │   ├── robots.txt.ts                => 爬虫文件
│   │   └── rss.xml.ts                   => RSS文件
│   ├── plugins             => 插件
│   ├── scripts             => 脚本
│   ├── styles              => 样式
│   ├── type                => 类型

- 然后创建新的项目，选择从GitHub导入

这样你在**Github King**上面做的每一个修改都会被同步到**Netlify**
### 发帖教程
首先创建一个md文档，在头部插入
:::
```md
---
title: 标题
categories: 分类
tags:
  - 标签1
  - 标签2
id: 文章ID
date: 文章创建日期
updated: 文章更新日期
cover: "封面图URL (为空默认随机内置封面 /public/assets/images/banner)"
recommend: false # 是否推荐文章
top: false # 是否置顶文章
hide: false # 是否隐藏文章
comment: false # 关闭页面评论（默认开启）
---
```
:::note
后面直接编辑你的正文内容就可以了，编辑好了之后直接通过**Github King**推送到`src/content/blog`，过2分钟就可以看到你的文章了
:::
